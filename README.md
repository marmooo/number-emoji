# Number Emoji

[An emoji drawing game to connect numbers](https://marmooo.github.io/number-icon/).

## Installation

1. `bash install-fonts.sh`
2. `bash build-svg.sh`
3. `deno run --allow-read --allow-write listup.js`

## Build

```
bash build.sh
```
